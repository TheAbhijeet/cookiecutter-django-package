# Django Package Cookiecutter Template

This repository contains a production-grade Cookiecutter template for generating reusable Django applications.

## Features

- **src/ Layout:** Prevents accidental imports.
- **Demo Project:** Standalone Django project for testing.
- **PyPI Ready:** Configured with `pyproject.toml`.
- **CI/CD:** GitHub Actions included.
- **Windows Compatible:** PowerShell instructions included.

## Prerequisites

- Python 3.8+
- Cookiecutter (`pip install cookiecutter`)

## Usage

```bash
cookiecutter .
```
