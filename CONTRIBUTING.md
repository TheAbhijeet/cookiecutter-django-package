# Contributing to { { cookiecutter.project_name } }

## How to Contribute

1. Fork the repository.
2. Create a branch: `git checkout -b feature/name`.
3. Make changes and run tests: `uv run pytest  -v`.
4. Commit and push.
5. Open a Pull Request.

## Coding Standards

- Follow PEP 8.
- Use type hints.
- Write tests for new features.

```
uvx prek
```

## Reporting Issues

Use GitHub Issues. Include steps to reproduce and environment details.
